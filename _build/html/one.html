Introduction to Database and Database Management Systems

This unit introduces the learner to data and databases. To begin with, this module defines what a dataset is, informs the learner about data, open data and different sources of data and how it impacts policy. Subsequently, the module introduces the different types of data and attributes. The learners will learn about the Data Pipeline, a step by step guide to working on a data driven project from start to end. It then focuses on a framework that makes it easy to work through various stages to generate useful insights or products from data. Next, learners will be introduced to the different formats of data, mainly focusing on open data formats.

Learning Objectives:

  • Identify the various stages of the Data Pipeline and how it can be used in data work.

  • Understand different types of data formats and distinguish between machine readable structured data and unstructured data..

  • Introduce to the different types of databases.

  • Understand relational databases and its applications.

  • Understand the value of data linkage.

Tip

The learners should feel free to skip modules that they are familiar with and jump to topics that are new to them. To test their understanding and engage with the materials deeply, the learners are encouraged to attempt the hands-on exercises and work through the discussion questions that are interspersed throughout the booklet.
